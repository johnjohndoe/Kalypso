Die Zeitreihen in diesem Verzeichnis werden für das SpreeModell benötigt. Da z.Z.
keine Zu- Abflüsse automatisch aus PSICompact kommen werden die hier angegebene
Zeitreihe als Input benutzt.

Die anderen benötigte Zeitreihen (Wasserstände, Niederschlagsmessungen) werden direkt aus der PSICompact
Schnittstelle geholt.
