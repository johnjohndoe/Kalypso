-------------------------------------------------------------------------
     Checkliste für die Erstellung der Kunden-CD für Sachsen / LHWZ
-------------------------------------------------------------------------

o Verzeichnisstruktur + bestehende Dateien anhand von cd-root erstellen

o Dokumentation/Feinkonzept
  folgende Dateien hinkopieren (aus dem Projektverzeichnis oder bestehende CD)
  - mapping_anforderungen_los2.xls
  - teil1-anwendungsfaelle_v2.1.doc
  - teil2-architektur_v2.1.doc
  - teil3-funktionsuebersicht_v2.1.doc
  - teil4-zuordnungstabelle_v2.1.doc

o Dokumentation/Handbücher
  dort folgende Pdf-Dateien erstellen (passende Sachsen-Launches benutzen)
  - Grafik-Benutzerhandbuch.pdf
  - Kalypso-Benutzerhandbuch.pdf
  - Kalypso-TechnischeDokumentation.pdf

o Dokumentation/Javadoc
  dort die Javadocs erzeugen. Am einfachsten die Eclipse
  Funktionalität dafür benutzen: Menü Project, Generate Javadoc...
  (in der Java Perspektive)

o Installation/Client/Grundinstallation
  beinhaltet schon der "minimalistic" Eclipse inkl. I18n als 
  gezippte Datei: minimalistic-eclipse.zip

o Installation/Client/KalypsoDateien
  dort ein KalypsoFeatureSachsen-Export machen und als 
  feature&plugins-sachsen.zip abspeichern

o Installation/Server/Grundinstallation
  folgende Dateien sind schon da:
  - j2sdk1.4.2_06.zip (Java SDK)
  - Tomcat5.0.zip (Tomcat Basisinstallation, schon für Kalypso vorbereitet mit:
    batch-Datei zum starten
    libs
    minimalistic webapps Verzeichnis)

o Installation/Server/KalypsoDateien
  die Ant-Build /KalypsoConfSachsen/deployable-tomcatstuff/build.xml benutzen
  - clean
  - psicompact-echte
  - deploy-lfug
  und dann die in /KalypsoConfSachsen/deploy erstellte Artefakten kopieren

o Quellcode
  KalypsoSachsenWorkspace
  KalypsoSachsenModelleWorkspace

o Werkzeuge
  dort Eclipse-SDK-3.0.2-win32.zip hinkopieren (liegt z.B. auf J:\hyd5309\software\eclipse)
  dort NLpack-eclipse-platform-3.0.x-win32.zip hinkopieren (liegt z.B. auf J:\hyd5309\software\eclipse)

o liesmich.txt-Datei anpassen, je nach dem was noch auf die CD kopiert wurde
