Output Results



User defines in preprocessing menu for simulation the directory which processor uses for simulation and the store outputs.
After successful simulation in the result folder several folders are generated. The folders: inp.dat, hydro.top, klima.dat, na-model, start are and xml_temp are generated automatical form processing routine to run simulation. The folder results_kalypso stores all simulation results, for the set nodes and catchments in theme ControlData:

OUTPUT DISCHARGE FOR NODES

For catchments following outputs are interesting for shortterm simulation in order to forecasting:
  • OUTPUT DISCHARGE FOR CATCHMENTS [m³/s]
  • Output Temperature [°C]
  • Output Precipitation [mm/h]
  • Output Snow Height [mm]
  • Output Soil Moisture Regime
  • OUTPUT SURFACE FLOW, NATURAL [m³/s]
  • OUTPUT SURFACE FLOW, SEALED [m³/s]
  • OUTPUT INTERFLOW [m³/s]
  • OUTPUT BASEFLOW [m³/s]
  • STORAGE VOLUME [hm³]
  • OUTPUT STORAGE [m³/s]
  • STORAGE HEIGHT OF WATERTABLE [m]
If numerous simulations run and result directory is not changed, Forecast Model creates for each simulation a new result folder (different suffix-numbers). For each simulation a log-files is generated which present main hydrological process values of waterbalance for the catchments or create error messages if simulation fails.

To evaluate results GraficTool is used which can be opened from Windows 2000 menu. When staring GraphicTool user selects directory results_kalypso and chooses from list the interested output file and catchment/ node.
Processor Kalypso RRM generates hydrological outputs for selected nodes and catchments in one file per kind of hydrological process or input time-series (precipitation and temperature).



GraphicTool offers functionalities to visualize outputs and create templates for calibration or forecasting.
Templates (filename.tpl) enables to fix the import of hydrographs customised layoutstyle of GraphicTool. Diagrams can be stored as *bmp to import them in papers or reports. Furthermore they can be published for flood information.
