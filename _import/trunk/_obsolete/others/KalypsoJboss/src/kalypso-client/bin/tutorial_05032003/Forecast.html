Forecast



Same workflow like simulation flood event. To forecast flood former steps before simulation are important:
  • Set forecast time-series if data can be required by weather report or other institutions in table forecast in DataCenter for contributing stations.
    If stations in subsystem of simulation receive no forecast precipitation or temperature only measured date will be used for imput.
  • Define connection on DataCenter forcast in tables RainStation and ClimateStation
  • Forecast values are mapped with measured time-series till end date of measured interval at station.
