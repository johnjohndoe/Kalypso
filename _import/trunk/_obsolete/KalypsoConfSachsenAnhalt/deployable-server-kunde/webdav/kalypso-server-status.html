Systemdienst

Die KALYPSO-Webdienste werden durch den Apache Tomcat Application Server zur Verfügung gestellt. Sollte ein oder mehrere KALYPSO-Webdienste nicht antworten, können Sie mit dem Neustart des Tomcat-Systemdienst versuchen, ein lauffähiger Systemzustand zu erreichen: öffnen Sie Systemsteuerung, anschliessend die Verwaltung und dann die Dienste. Dort suchen Sie den Eintrag Apache Tomcat KalypsoServer und starten Sie der Dienst erneut (entweder über Neu starten oder über Beenden und später wieder Starten).

Prozess

Ein Prozess namens tomcat5.exe muss in die Prozessliste des Task-Manager sichtbar sein wenn der Tomcat Application Server richtig gestartet wurde.
