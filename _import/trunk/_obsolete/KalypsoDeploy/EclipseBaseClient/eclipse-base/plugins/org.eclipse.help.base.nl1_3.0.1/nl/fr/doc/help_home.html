Utilisation du système d'aide Eclipse

Parcourez la liste des rubriques dans le cadre Table des matières () situé sur la gauche. Cliquez sur une rubrique pour l'afficher. Utilisez les boutons Précédent et Suivant pour naviguer dans l'historique des rubriques consultées.

Recherche

Pour trouver rapidement les rubriques d'aide sur un sujet particulier dans la documentation, entrez une requête dans la zone Rechercher. Utilisez le cadre Recherche () pour afficher la vue Recherche. Vous pouvez limiter l'étendue de la recherche en ne sélectionnant que les sections qui vous intéressent.

Synchronisation

Après avoir lancé une recherche et identifié la rubrique que vous souhaitez consulter, cliquez sur le bouton Régénérer/Afficher la rubrique en cours () ou le bouton Afficher dans la table des matières () pour faire correspondre l'arborescence de navigation et la rubrique en cours. Il peut également vous être utile d'effectuer une synchronisation après avoir suivi des liens contenus dans une rubrique.

Capacités

Pour afficher la documentation sur les capacités désactivées dans l'application, cliquez sur le bouton Afficher toutes les rubriques (). Lorsque vous décidez d'afficher toutes les rubriques dans la table des matières, les titres de la documentation relatives aux capacités désactivées s'affichent dans la table des matières et les résultats de la recherche.

Pour en savoir plus

Si vous ne trouvez pas de réponse à votre question dans l'aide en ligne, visitez le site eclipse.org pour rechercher des articles et prendre part à la communauté Eclipse.
