使用 Eclipse 說明系統

在左邊的「內容」頁框中瀏覽主題 ()。 按一下主題以讓它顯示。使用「回上頁」與「到下頁」按鈕 來導覽至檢視過的主題記錄。

搜尋

若要在說明文件中快速找出特定項目的主題，請在「搜尋」欄位中輸入查詢。 使用「搜尋」頁框 () 來顯示「搜尋」視圖。 您可以縮小搜尋的範圍，方法是僅選取您有興趣的區段。

同步化

在您執行搜尋，並找到您想要找尋的主題之後，按一下， 「重新整理/顯示主題」 按鈕() 或「顯示在目錄中」按鈕()， 來將導覽樹與目前的主題相配對。在進入主題內的連結之後，同步化也是很有用的。

功能

如果要顯示應用程式中所停用之功能的相關文件，請選取 「顯示所有主題」按鈕()。 當您選擇在目錄中顯示所有主題時，任何已停用的活動的相關文件標題都會顯示在目錄中，也會出現在搜尋結果中。

詳細資訊

如果您無法從線上說明中得到問題回答， 請參閱我們的網站 eclipse.org， 來尋找相關文章以及加入 Eclipse 社群。
