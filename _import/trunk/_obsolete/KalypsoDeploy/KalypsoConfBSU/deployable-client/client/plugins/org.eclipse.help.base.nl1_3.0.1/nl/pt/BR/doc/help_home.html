Utilizando o Sistema de Ajuda do Eclipse

Procure tópicos na estrutura Conteúdo () à esquerda. Clique em um tópico para exibi-lo. Utilize os botões Voltar e Avançar para navegar no histórico dos tópicos visualizados.

Pesquisando

Para localizar rapidamente os tópicos em um assunto específico na documentação, digite uma consulta no campo Procurar. Utilize a estrutura Procurar () para exibir a visualização Procurar. Você pode restringir o escopo da procura, selecionando somente as seções em que estiver interessado.

Sincronizando

Depois de executar uma procura e localizar o tópico que estava procurando, clique no botão Atualizar / Mostrar Tópico Atual () ou no botão Mostrar no Índice () para fazer a correspondência entre a árvore de navegação e o tópico atual. Você também pode considerá-lo útil para sincronização após os links seguintes no tópico.

Recursos

Para mostrar a documentação sobre os recursos que são ativados no aplicativo, selecione o botão Mostrar Todos os Tópicos (). Ao escolher mostrar todos os tópicos no índice, os títulos para a documentação sobre as atividades desativadas são mostradas no índice e nos resultados da procura.

Mais Informações

Se você não encontrar a resposta para a sua pergunta na ajuda on-line, visite o Web site no eclipse.org para localizar os artigos e participe da comunidade do Eclipse.
