Eclipse 도움말 시스템 사용

왼쪽의 컨텐츠프레임()에 있는 항목을 찾아보십시오. 표시할 항목을 누르십시오. 표시된 항목의 히스토리 내에서 탐색하려면 이전 및 다음 단추를 사용하십시오.

검색

문서에서 특정 주제에 대한 항목을 신속하게 찾으려면 검색 필드에 조회할 내용을 입력하십시오. 검색 보기를 표시하려면 검색 프레임 ()을 사용하십시오. 관심있는 섹션만 선택하여 검색의 범위를 좁힐 수 있습니다.

동기화

찾기를 실행하고 있는 주제를 찾은 후에, 현재 주제 표시 / 새로 고치기 단추() 또는 목차 표시 단추() 를 눌러 탐색 트리를 현재 주제와 일치시키십시오. 또한 항목 내에서 링크를 따라간 후에 동기화시키는 것이 유용할 수도 있습니다.

성능

응용프로그램에서 사용 불가능한 성능에 대한 문서를 표시하려면 모든 주제 표시 단추 ()를 선택하십시오. 목차의 모든 주제를 표시하도록 선택하면 사용 불가능한 모든 활동에 대한 문서의 표제가 목차에 표시되고 검색 결과에도 표시됩니다.

자세한 정보

온라인 도움말에서 질문에 대한 응답을 찾을 수 없는 경우, 웹 사이트 eclipse.org에서 관련 주제를 찾아보거나 Eclipse 커뮤니티에서 확인할 수 있습니다.
