KALYPSO_JRE (damit die builder in einer separaten jre laufens)
===========
In 'Preferences / Java / Installed JREs ' eine neue JRE einrichten 
mit Namen 'KALYPSO_JRE'. Es muss eine jre 1.4.2 sein, am besten
die jdk 1.4.2.06 (Tip: eine JDK beinhaltet immer eine JRE, Vorteil von
JDK: Debug-Möglichkeit).
