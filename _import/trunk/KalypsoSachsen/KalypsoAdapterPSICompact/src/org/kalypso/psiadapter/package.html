Beinhaltet die Klassen um die Schnittstelle von PSI auf die interne Kalypso Schnittstelle anzupassen. Es gibt zur Zeit (Testphase, PSI-Durchtisch) folgende Menü-Funktionen:
  • DistributeFile: eine Datei wird in PSICompact registriert
  • RemoveFile: eine Datei wird von PSICompact gelöscht
  • WriteProtocol: ein Text wird in das PSICompact Protokoll geschrieben
