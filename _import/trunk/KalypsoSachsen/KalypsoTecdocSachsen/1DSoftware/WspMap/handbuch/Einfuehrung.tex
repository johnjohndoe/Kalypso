\section{Einführung}
Das Programm \wspmap\ dient der Erstellung und Visualisierung von Projekten zur eindimensionalen Wasserspiegellagenberechnung mit dem Programm \wspwin.\\
Das vorliegende Programm ermöglicht die Ein- und Ausgabe der im Programm  \wspwin\ verwendetens hydraulischen Daten in Form von Lageplänen (Karten).\\
Aus dem mit \wspwin\ berechneten Wasserspiegellagen können mit Hilfe eines Digitalen Geländemodells (DGM) Überschwemmungsflächen und neuerdings auch Fließtiefen generirt werden.\\
Ferner bietet \wspmap\ die Möglichkeit, vorhandene Querprofile weiter zu beachten (Verlängerung der Vorländer) bzw. zusätzliche Querprofile in ein Projekt einzufügen. Durch die Import - Funktion von Vermessungsdatensätzen ist es möglich, Projekte neu zu erstellen und anschließend die Berechnung mit \wspwin\ vorzunehmen.\\
Durch die Einbindung von ESRI Map - Opjects 2 entspricht \wspmap\ der Konzeption der ESRI - Produkten Aice Vier und ArcExplorer.\\
Beim Einlesen der \wspwin\ - Datensätze werden somit automatisch ESRI - konpatible Daten (Shape - Files) erzeugt. Entsprechend lassen sich externe Daten in \wspmap\ visualisieren und teilweise bearbeiten.\\\\
Im einzelnen beinhaltet \wspmap\ die folgenden Funktionen:
 
\begin{itemize}
	\item 
	Visalisierung
	
		\begin{itemize}
			\item
			Darstellung von georeferenzierten \wspwin\ - Querrprofilen, inklusiveder Fließzonen (Trennflächen, Bordrollpunkten, Durchström Bereichen) im Lageplan.
			\item
			Hinzuladen von weiteren Informationen, wie Hintergrundkarten, Luftbilder oder beliebigen Shape-Files / dxf - Files
			\item
			Import von Vermessungsdaten (da66, da65, tripple) und Erzeugen von \wspwin\ - Datensätzen
		\end{itemize}
		
	\item 
	Datenbearbeitung im Lageplan
	
		\begin{itemize}
			\item 
			Verschieben der Fließzonen (Trennflächen, Durchströmbereiche usw.)
			\item
			Verlängern der Profile (Vorlandbereiche) mit Hilfe eines digitalen Geländemodells (DGM)
			\item
			Erzeugen zusätzlicher Querprofile (DGM notwendig)
			\item 
			Zuweisung von Rauheiten über Nutzungsklassen 
			\item
			Speichern der Änderungen und Weiterverarbeitung  (Berechnung) in \wspwin .							\end{itemize} 		
			
	\item
	Weiterverabeitung  von Berechnugsergebnissen
	
		\begin{itemize}
			\item 
			Darstellung berechneter Wasserspiegellagen (als Punkte im Querprofil)
			\item
			Erzeugen von Überschwemmungsflächen durch Verschiedene Wasserspiegel mit einem DGM
			\item
			Ermittlung und Darstellen von Fließtiefen (Verechnung von Wasserspiegellagen mit DGM)
			
			$\Longrightarrow$ Ausgabe (Plot) von Überschwemmungs- und Fließtiefenkarten
		\end{itemize}
		
\end{itemize}
