\section{Standardlayouts erstellen und bearbeiten}
\label{sec:templates}
Um nicht jeden Plot einzeln formatieren zu müssen, können auch Standardlayouts erstellt
werden, die dann relativ einfach auf neue Plots übertragen werden können. Wählen Sie hierzu
\menu{Extras / Standard} (Abb. \ref{fig:standardeinstellungen}).
Wenn Sie einen neuen Standard erzeugen möchten, wählen Sie \ctrl{Hinzufügen}. Es
erscheinen zwei Editierfelder, in denen Sie zum einen nach Klicken in das Einstellungen-Feld
(links) einen Namen für Ihr Layout vergeben können (z.B. LfW-Querprofil) und zum anderen
einen Stempel zuordnen können. Sobald Sie in das Stempelfeld klicken erscheint ein
Listenfeld mit allen verfügbaren Stempeln. Wenn Sie eine Standardeinstellung markieren und
anschließend \ctrl{Bearbeiten} drücken, kommen Sie wieder zum Dialogfeld \dialog{Eigenschaften}.
Hier können Sie sämtliche Einstellungen wie unter \ref{sec:profildarstellung}
erläutert selektieren und dem Standard zuordnen. Beim späteren Öffnen eines neuen
Plots / Profils wird als Default der Standard gewählt, dessen Kontrollkästchen aktiv geschaltet
wurde. Es besteht auch die Möglichkeit keinen Standard vorzudefinieren (alle Kästchen
ausgeschaltet). Um einem einzelnen Plot einen Standard zuzuordnen wählen Sie in der Karte
\dialog{Allgemein} unter \ctrl{Einstellungen / Eigenschaften} das gewünschte Layout aus.
