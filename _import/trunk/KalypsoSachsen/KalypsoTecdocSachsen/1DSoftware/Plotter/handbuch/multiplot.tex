\section{Mehrere Plots auf ein Blatt}
\label{sec:multiplot}

Wird beim Laden der Profildaten die Option \ctrl{Mehrere Plots auf ein Blatt} selektiert (siehe Abbildung~\ref{fig:auswahl-profile}), so werden alle ausgewählten Profile in einem Plotter-Fenster geöffnet. Die Profile werden dabei mit der vorgegebenen Standardvorlage (siehe Abschnitt~\ref{sec:templates}) formatiert und Zeilenweise angeordnet. Ist in der Standardvorlage ein Stempel ausgewählt, so wird dieser rechts unten plaziert.

\begin{figure}[htbp]
	\centering
	% FEHLT!
	%\includegraphics[width=1.0\textwidth]{multiplot} 
	\caption{Mehrere Plots auf einem Blatt}
	\label{fig:multiplot}
\end{figure}

Passen nicht alle Profile auf ein einzelnes Blatt (zum Einstellen der Seitengrösse siehe Abschnitt~\ref{sec:print_plot}), so werden wie im Plotter üblich mehrere Blätter erstellt und die Blattgrenzen durch blaue Linien dargestellt. Passt ein Profil nicht vollständig auf ein Blatt, so wird es komplett weggelassen und eine entsprechende Meldung ausgegeben.

Prinzipiell stehen im Modus 'mehrere Plots auf ein Blatt' die gleichen Bearbeitungsmöglichkeiten zur Verfügung wie im normalen Modus. Lediglich die Profildarstellung wird ausschliesslich über die Standardvorlage gesteuert und kann nachträglich nicht mehr verändert werden. Statt dem entsprechenden Dialog im Einzelprofilmodus steht durch das Menü \menu{Bearbeiten / Eigenschaften} ein Dialog zum Einstellen der Abstände zwischen den Profilen und zum Ändern der Blattüberschrift zur Verfügung.
