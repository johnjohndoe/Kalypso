\section{Zeichnungen speichern und exportieren}

Die Plots können jederzeit über \menu{Datei / speichern} gesichert werden. Die Namensvergabe erfolgt dann automatisch. über \menu{Datei-speichern unter} haben Sie die Möglichkeit, selbst einen Dateinamen zu vergeben. So können Sie z.B. auch mehrere unterschiedliche Plots für eine Profildatei anlegen, die Sie dann auf die soeben beschriebene Weise (nach vorherigem \menu{Projekt-schliessen} ) wieder öffnen können.
Es ist zu empfehlen, die Plots häufiger zwischendurch zu speichern, damit aufwendige Formatierungen nicht Gefahr laufen, verloren zu gehen. 

\hypertarget{cmd:datei-dxf-export}{}
Alternativ können Sie über \menu{Datei / Dxf Export} eine dxf-Datei erstellen, die Sie in einem CAD-Programm
einlesen und ggf. weiter aufbereiten können.
