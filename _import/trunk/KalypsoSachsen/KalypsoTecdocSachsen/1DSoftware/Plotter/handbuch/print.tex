\section{Drucken / Plotten}
\label{sec:print_plot}
Bevor Sie die Zeichnung zum Drucker bzw. Plotter schicken, haben Sie noch weitere
Möglichkeiten, die Seite einzurichten.
Innerhalb einer Arbeitssitzung mit \wspplot\ können Sie eine Standard Seiten-
/Druckereinrichtung über \menu{Datei / Druckereinrichtung (Standard) } festlegen (Abb. \ref{fig:seite-einrichten}).
Hier wählen Sie den zu verwendenden Drucker sowie die Papiergröße aus und legen die
Ränder (Abstand Papierrand-Rahmen) fest. Nachdem Sie die Standarddruckereinrichtung
festgelegt haben, werden alle Profile, die Sie danach öffnen, automatisch mit dieser
Standarddruckereinrichtung formatiert. Genau die gleichen Einstellungen lassen sich unter
\menu{Datei / Seite einrichten} für einen Einzelplot festlegen. Die Standardeinstellungen
bleiben nur so lange gültig, wie das Programm nicht wieder geschlossen wird.

\begin{figure}[hbtp]
	\begin{center}
		\includegraphics[scale=0.70]{seite-einrichten}
	\end{center}
	\caption{Seite einrichten}
	\label{fig:seite-einrichten}
\end{figure}

Über \menu{Datei / Drucken} schicken Sie den Plot zu ihrem Ausgabegerät.
