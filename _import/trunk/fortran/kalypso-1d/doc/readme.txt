
Aktuelle KALYPSO-1D Version:
============================
Version 2.1.4.0, 23.07.2008


Installation eines neuen Rechenkerns für WSPWin
-----------------------------------------------
1.) Die aktuelle Kalypso-1D.exe in das 
    WSPWin-Programmverzeichnis kopieren. 
    (Beispiel: C:\Programme\BCE\WspWin ...)

2.) Den Namen der aktuellen *.exe in die erste
    Zeile der Konfigurationsdatei "wspbce.cfg"
    eintragen. Die Konfigurationsdatei liegt ebenfalls
    im WSPWin-Programmverzeichnis 
    (Beispiel: C:\Programme\BCE\WspWin ...)
