使用 Eclipse 帮助系统

浏览左边目录框架（）中的主题。单击某个主题以显示它。使用后退和前进按钮来浏览已查看的主题的历史记录。

搜索

要在文档中快速找到有关特定内容的主题，在搜索字段中输入查询。使用搜索框架（）来显示“搜索”视图。可以通过只选择感兴趣的部分来缩小搜索的范围。

同步

在运行搜索并找到要查找的主题之后，单击刷新／显示当前主题按钮（）或在目录中显示按钮（）来使导航树与当前主题相匹配。在访问主题中的链接之后您可能也会发现同步是很有用的。

功能

要显示有关在应用程序中禁用的功能的文档，可选择显示所有主题按钮（）。当选择显示目录中的所有主题时，有关任何已禁用活动的文档的标题就会显示在目录中，还会出现在搜索结果中。

更多信息
