SimulationCase


SimulationCase theme combines spatial and temporal data.
For SimulationCase context menu of rows offers command for RUN SIMULATION.
For the selected combination RRM starts Processing dialog.
To choose model and control version the id (visualised in Project Tree for each structure element) is used to identify versions for simulation.
