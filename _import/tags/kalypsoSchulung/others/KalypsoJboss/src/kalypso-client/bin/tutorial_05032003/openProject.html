Start/End Project


In the Windows 2000 start menu FLOMATIS Forecast System appears. Four programs can be executed:
  • Kalypso RRM
  • Kalypso Forecast
  • DataCentre
  • GraficTool
Starting Kalypso RRM Application Window opens:



Toolboox PROJECT VIEW opens automatically and user can choose in menu bar DATA:
  • CONNECT DATABASE
  • CREATE VERSION
  • UPDATE TREE
Without connection to database mentioned PROJECTTREE in PROJECTVIEW is empty. CONNECT TO DATABASE load data to system.
Events from DB update continuously data but user can manually via UPDATE TREE refreshing tree and get actual state of versioning. Create version enables user to create a new elements of ProjectTree


CREATE VERSION
enables user to create a new structure elements of PROJECTTREE: Project element structures data dependent on topological region because in DB only one table for all thematic data exists.
A project can set if at least one version of a theme is set. As consequence of pilot system for UTRB a new version of Modeldata can be created but the preprocessing to accomplish data sets is not implemented in program. VIEW in menu bar presents logging output of simulation runs, if simulation is done without errors: simulation successful appears at the end.







To end the program click on standard window close button in header.
Actual stady of editing is consistently saved like explained in Chapter about DB.
