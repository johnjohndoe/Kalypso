Dieses Paket dient dazu auf Resourcen über eine Beschreibung ('source', im Allgemeinen ein beliebiger String) zuzugreifen.

Das eigentliche Laden (ggfls. auch wieder speichern) einer Resource wird durch Instanzen der Klasse {@link org.kalypso.commons.loader.ILoader} durchgeführt.

Auf die Loader selbst wird über die {@link org.kalypso.commons.loader.ILoaderFactory} zugegriffen
