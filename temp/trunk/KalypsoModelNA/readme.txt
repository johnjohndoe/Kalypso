Dieses Projekt enthält den Code zur Einbindung des Niederschlag Abfluss
 Modelles Kalypso-NA. Dieses Modell entspricht der Weiter entwiklung
 des NA-Modelles BCENA.
 Es wird zur Zeit in folgenden Projekten (in Kombination
 mit diesem user interface) verwendet:
 - LHWZ-Sachsen (Modell Weisse Elster)
 - Behörde für Stadtentwicklung und Umwelt Hamburg (Kollau, Falkengraben)
 - versch. Ingenieurbüros (Oberalster, Berner Au, Tarpenbek)
 - Lehre (Obere kollau, Quellgebiet Stör)
 - E-Learning DAAD (Quellgebiet Stör)
 
 Es bestehen außerdem folgende Testprojekte:
 - BasicTestModel (zur Parameterstudie)
 - AdvancedTestModel (zu Testzwecken des user interfaces)
